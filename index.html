TOOL, INSTRUCTIONS, IDEA

Shiftable Type is an intuitive typesetting tool developed by Hannes Altmann that lets you experiment with indentation of text lines. Designed for use ahead of an editorial design project, it helps generate inspiration and quickly visualize layout ideas.

Click on a tile to select it. Click it again to deselect. You can select multiple tiles in a row by clicking and dragging across them horizontally, then releasing the mouse button. To deselect multiple tiles, use the same gesture on already selected ones. To move a tile or an entire row horizontally along the grid, hold Shift + Click on it while dragging left or right. If you want to select an entire block of tiles, hold Space and draw a selection rectangle with your mouse. Release to confirm the selection.

My tool is intended to be used to play intuitively with typesetting (particularly with indentations of text lines). It mimics the functionality of the abacus, the interface is designed very minimal and straightforward to guarantee an experimental user experience. The output can be seen as „templates“ of experimental (or conventional) text line arrangements that can serve as inspiration for your next editorial typesetting project.

The idea for the tool really came from a personal desire to have an environment where I could test, experiment and try out new things within editorial typesetting and to be able to do this as fast, intuitive and effortless as possible (unlike in InDesign). For one, the realization is inspired by the mechanics of the abacus but also by manual typesetting methods from the letterpress era, when movable type (metal sorts) was composed by hand on a composing stick for each page by workers called compositors.
